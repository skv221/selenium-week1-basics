## Prerequisites

- IDE like VS Code or PyCharm  

## How to Run  
1. Clone the repository:  
   ```bash
   git clone <repository-url>
   cd selenium-week1-basics
   ```
2. Install dependencies:  
   ```bash
   pip install selenium
   ```
3. Run the script:  
   ```bash
   python main.py
   ```

- The console will print the page title and the first heading.
